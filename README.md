## 2.项目功能

 - a.用户管理
用户的注册，登录，退出。注册成功时需要将用户信息写入数据库，并在HDFS中建立用户的专属目录。
 - b.文件管理
提供文件上传，下载，删除等功能。提供目录的管理，可以创建目录，复制或移动文件或目录到指定目录下面；可以删除目录。
 - c.界面设计
为系统设计并实现主界面，基于窗口或基于浏览器均可。所有操作均在人机交互界面中完成。
